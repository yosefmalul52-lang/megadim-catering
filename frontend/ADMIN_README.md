# הפעלת דשבורד הניהול

## הפעלה ידנית

להפעלת דשבורד הניהול על port 4201:

```bash
cd frontend
npm run start:admin
```

לאחר מכן, פתח דפדפן וגש ל:
**http://localhost:4201**

## הערות חשובות

1. **הדשבורד רץ על port נפרד (4201)** - לא על אותו port של האתר הציבורי (4200)
2. **האתר הציבורי רץ על port 4200** - לא מכיל את הדשבורד
3. **הדשבורד נגיש רק דרך port 4201** - לא דרך הנתיב `/admin` באתר הציבורי

## פתרון בעיות

אם אתה מקבל שגיאת `ERR_CONNECTION_REFUSED`:
1. ודא שהשרת רץ: `lsof -i :4201`
2. בדוק את הטרמינל לשגיאות
3. נסה להריץ שוב: `npm run start:admin`
